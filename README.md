stream-ciphers-go
=================

Selfmade stream ciphers for go


```sh
# get SexySecret
go get github.com/maxymania/stream-ciphers-go/sexysecret

# get PureChaos1
go get github.com/maxymania/stream-ciphers-go/purechaos1
```

// import SexySecret
